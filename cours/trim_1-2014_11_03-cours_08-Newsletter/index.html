La méthode Fireworks

Rajoutez les textes et ajoutez le style de chaque texte sur son <td> le contenant. Utiliser l'attribut "valign" pour choisir l'allignement vertical du texte dans sa cellule.

Envoyer ses propres mails

Un script PHP vous est fourni avec ce cours dans le dossier "send_email". Il ne fonctionne qu'avec les comptes gmail.

Lancez votre serveur apache local (MAMP ou WAMP par exemple).

Envoyer ses propres mails

Ouvrez index.php dans le dossier "send_email" avec un éditeur de code et renseignez les différentes variables ($to, $gmail_username, $gmail_password, ...).

$newsletter correspond au chemin relatif vers votre newsletter.

Envoyer ses propres mails

Lancez le script php en vous rendant à son adresse avec votre serveur local (exemple http://localhost/newsletter/send_email/index.php)

La page vous indiquera "Sent !" si tout s'est bien passé.

Sinon vous aurez un message indiquant l'erreur.
