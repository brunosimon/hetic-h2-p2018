Bonjour Monsieur ! Vous allez bien ?

Mon devoir sur le thème du sondage prend la forme d'une éléction pour un président.

Je demande quelques informations au moment du vote afin de faire des statistiques et afficher des graphiques.

Chaque graphique affiche des statistiques sur l'éléctions, ils prennent en compte une multitude de critères.

Les technologies utilisées sont : 
Javascript
Jquery
Ajax
Google Visualization API
Php
MySQL

Les formats utilisés sont : 
HTML/CSS
Json


Fichiers :
chart_database.sql -> La base de donnée
config.php -> Le fichier de configuration pour me connecter a la base de donnée
index.php -> Page où l'on rempli le formulaire (sondadge)
result.php -> Page de résultat du vote, avec 9 graphiques differents
result.js -> Fichier qui contient les scripts de générations des graphiques
getdata.php -> Fichier appelé en Ajax par result.js qui contient l'ensemble des requètes des graphiques à la base de donnée
index.css -> Le css de index.php
result.css -> Le css de result.php

Je vous souhaite une bonne lecture de mon code et des graphiques !
