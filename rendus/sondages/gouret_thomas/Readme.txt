Voila mon exercice

La sauvegarde de ma base de donnée se trouve dans le dossier quizz. 
Elle permet de faire fonctionner mon site Internet composé de 5 pages.
Sur la première, on explique le sondage et on clique pour commencer.
A la suite de cela, on est soumis successivmentt à un choix entre deux produits, et l'on vote pour celui que l'on préfère.
Sur la dernière on voit apparaitre, tous les produits avec leur nombres de vote respectifs classés dans l'ordre décroissant.
