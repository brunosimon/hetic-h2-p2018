Readme File

Informations pratiques

  • Un compte administrateur est créé de base et occupe la permière entrée de la table "users"
    • Pseudo : admin
    • Password : admin

Actions possibles

  • Connexion et inscription
  • Remplir le questionnaire si connecté
  • Voir la liste des membres si administrateur
  • Consulter son compte et changer son mot de passe

Problèmes éventuels

  • Dans la table users, la ligne 1 est celle de l’administrateur, je n’ai pas réussi faire une selection sur ma table si le rang est différent de la valeur "admin" donc je fais une selection sur toutes mes lignes d’id supérieur à la première ligne
  • Le style css n'est plus pris en compte quand on met un "/" à la fin de l'url. Par exemple : ./sign/

Quelques explications

  • Chemin suivi lors de la connexion
    • redirection automatique sur la page d’administrateur
    • si on est pas au rang d’admin alors on sera redirigé sur la page de compte.
  • Si on est déjà connecté et qu’on tente d’accéder à la page de connexion/inscription, on est redirigé sur la page de compte
