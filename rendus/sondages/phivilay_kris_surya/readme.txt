Devoir PHP Sondage


Pour ce devoir, j'ai choisi de construire un site ressemblant à viedemerde.fr mais pour les personnes travaillant en agence web.

Le principe est simple, l'utilisateur peut lire des anecdotes et décider de voter ou non si celles-ci lui ont plu ou non par le biais d'un système de vote. Un like ou un dislike pour être plus précis. Le système se repose comme les vidéos youtube avec une barre de "progression" des votes où l'on peut apprécier visuellement les différents votes des utilisateurs.

Sur le site, on peut décider de voter selon le compte de Paul, Mathieu ou Bruno. Un utilisateur non-enregistré se verra l'interdiction de voter.

Dans la base de donnée, la date de vote de chaque utilisateur est enregistrée.

J'ai utilisé divers ressources comme jquery, ajax, javascript notamment pour la mise à jour en temps réel des votes.
Le reste est du simple HTML-CSS agrémenté évidemment de PHP.
