Bonjour,

Mon site est orienté sur le dernier jeu Blizzard en date : Heroes Of The Storm (ou HOTS).

Ce MOBA permet de combattre avec un grand nombre de combattants, nombre qui va encore continuer de croitre.

Afin d'aiguiller les joueurs pour trouver le Héros correspondant le plus à leur style de jeu, l'utilisateur doit répondre à une série
de questions sous forme de QCM/QCS. Puis, le ou les Héros correspondant à ces attentes s'affichent, avec le lien correspond à la page
de description détaillée / guides du Héros sur le site millenium.org.

Pour accéder au QCM (et aux statistiques que nous verrons juste après), il est nécessaire de se connecter au site ou de s'inscrire
le cas échéant. Une redirection automatique au niveau des pages QCM et statistiques a été mise en place.

La page de statistiques nous permet de voir le nombre de fois que chaque Héros a été proposé à un joueur, le nombre de fois que 
l'utilisateur a effectué le QCM ainsi que le nombre total de Héros retournés par le site.

La page d'acceuil présente les différents héros, ainsi que leurs fonctions et univers d'origine.

Features :
Inscription sécurisée et stockage en BDD
Login
Déconnexion
Redirection
Page 404
QCM sécurisée et traité pour une récupération ciblée dans une BDD en fonction des réponses du QCM
Contenus HTML générés dynamiquement (par exemple page d'accueil ou de statistiques)
Routing
Animations CSS pour un peu de gaïté
